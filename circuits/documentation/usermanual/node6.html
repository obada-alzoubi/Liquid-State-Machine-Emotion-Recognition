Getting and Installing Circuit-Tool

Circuit-Tool is distributed under the GNU General Public License and can be downloaded from http://www.igi.tugraz.at/circuits.

To install Circuit-Tool perform the following steps:

  1. Donwload Circuit-Tool from www.igi.tugraz.at/circuits
  2. Unzip the file circuits-VER.zip where VER stands for the version you have downloaded.

    This will create a subdirectory lsm and lsm/circuits

  3. Start Matlab and change into the directory lsm

  4. Run the Matlab script install.m.

  5. Add the path lsm to the Matlab search path; e. g.
    • addpath('/home/jack/lsm')} or
    • addpath('C:\Work\Neuroscience\lsm').

  6. Change into the directory lsm/circuits/demos and play around with them.

  7. Have fun using Circuit-Tool !
