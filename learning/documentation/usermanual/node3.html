Learning-Tool User Manual

Features of the current version

Matlab interface
Learning-Tool is mainly written in Matlab and hence

Runs under Unix (Linux) and Windows

Object oriented design
All the Learning Algorithms, target functions, and readouts are implemented as Matlab classes.

Parallel Processing
Usually the training of a readout involves many simulations of the same microcircuit with different inputs and training of several readouts. This can easily be run in parallel. Learning-Tool provides means to do this in a convinient way (by employing the Parallel Matlab Toolbox [Svahn, 2001]).
